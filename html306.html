정의(Definition)

  • <ol> 태그(ordered list)는 순서가 있는 목록형 태그입니다.
  • <ol> 태그는 순서가 있는 목록은 숫자 또는 알파벳으로 표현 할 수 있습니다.
  • <ol> 태그 type 속성을 통해 숫자, 문자, 로마 숫자들을 표현할 수 있지만, CSS list-style-type 속성을 통해 사용하는 것을 권장합니다.
  • <ol> 태그는 리스트 항목을 나타내는 <li>와 같이 사용됩니다.

비교(Compare)

  • <dl> 태그는 용어를 설명하는 목록형 태그입니다.
  • <dt> 태그는 용어의 제목을 설명하는 목록형 태그입니다.
  • <dd> 태그는 용어를 설명하는 태그입니다.
  • <ul> 태그는 순서가 없는 몰록형 태그입니다.
  • <ol> 태그는 순서가 있는 목록형 태그입니다.
  • <li> 태그는 목록의 항목 태그입니다.
