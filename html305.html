<object>

<object> 태그는 웹 문서에 포함된 멀티미디어 요소를 정의합니다.

특징 설명
카테고리 플로우 콘텐츠(Flow Content), 프레이징 콘텐츠(Phrasing Content), 임베디드 콘텐츠(Embedded Content), 팰퍼블 콘텐츠(Palpable Content)
속성 글로벌 속성 적용(Global Attributes), 이벤트 속성 적용(Event Attributes)
요소 블록 요소(Block Element)
닫는 태그 닫는 태그 사용(<object> ~ </object>)
버전 HTML4
시각적 표현 -

정의(Definition)

  • <object> 태그는 웹 문서에 포함된 멀티미디어 요소를 정의합니다.
  • <object> 태그는 <body> 요소 안에 설정합니다.
  • <object> 태그는 오디오, 비디오, 자바 애플릿, 액티브X, PDF, 플래시 파일을 지원합니다.
  • <object> 태그는 data 또는 type 속성이 포함되어야 합니다.

크로스브라우징(CrossBroswing)

  • HTML4에서 <object> 태그는 <body> 요소 안에만 설정 가능했으나, HTML5에서는 <head> 또는 <body> 요소안에 설정 할 수 있습니다.
