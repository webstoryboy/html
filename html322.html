<script>

<script> 태그는 스크립트를 정의합니다.

특징 설명
카테고리 플로우 콘텐츠(Flow Content), 프레이징 콘텐츠(Phrasing Content), 메타데이터 콘텐츠(Metadata Content)
속성 글로벌 속성 적용(Global Attributes), 이벤트 속성 적용(Event Attributes)
요소 인라인 요소(Inline Element)
닫는 태그 닫는 태그 사용(<script> ~ </script>)
버전 HTML4
시각적 표현 -

정의(Definition)

  • <script> 태그는 스크립트 문을 포함하거나 외부 스크립트를 연결할 수 있습니다.
  • 스크립트를 지원하지 않는 브라우저 사용자를 위해서 <noscript>를 설정합니다.

크로스브라우징(CrossBroswing)

  • HTML4에서는 type 속성이 필수이지만, HTML5에서는 선택사항입니다.
  • HTML5에서는 async 속성이 새로 생겼습니다.

속성(Property)

속성 값(예) 설명 버전
async async 외부파일 연동시 비동기 방식을 설정합니다.
charset charset = "UTF-8" 외부 파일의 문자 인코딩 방식을 설정합니다. -
defer defer 페이지가 완료되면 외부 파일을 실행시킵니다. -
src src = "URL" 외부 파일 주소를 설정합니다. -
type type = "text/javascript" 외부 파일의 형식을 설정합니다. -
xml:space preserve 외부 파일 연동을 설정합니다.
