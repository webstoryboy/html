<fieldset>

<fieldset> 태그는 폼 요소의 그룹 내부 영역을 설정합니다.

특징 설명
카테고리 플로우 콘텐츠(Flow Content), Dectioning Root, Listed, Form-associated element, 섹션 콘텐츠 (Sectioning content)
속성 글로벌 속성 적용(Global Attributes), 이벤트 속성 적용(Event Attributes)
요소 인라인 요소(Inline Element)
닫는 태그 닫는 태그 사용(<fieldset> ~ </fieldset>)
버전 HTML4
시각적 표현 테두리 생김

정의(Definition)

  • <fieldset> 태그는 폼 요소의 여러 컨트롤과 <label>을 그룹화 하는데 사용합니다.

크로스브라우징(CrossBroswing)

  • HTML5에서 disabled, form, name 속성이 추가되었습니다.

속성(Property)

속성 값(예) 설명 버전
disabled disabled fieldset 영역을 사용하지 않도록 설정합니다.
form form="id" fieldset 영역의 폼 아이디를 설정합니다.
name name="text" fieldset 영역의 이름을 설정합니다.
