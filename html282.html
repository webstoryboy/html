<html>

<html> 태그는 웹 문서의 최상위 요소 문서의 루트를 나타냅니다.

특징 설명
카테고리 -
속성 글로벌 속성 적용(Global Attributes), 이벤트 속성 적용(Event Attributes)
요소 -
닫는 태그 닫는 태그 사용(<html> ~ </html>)
버전 HTML4
시각적 표현 -

정의(Definition)

  • <html> 태그는 웹 문서의 최상위 요소 문서의 루트를 나타냅니다.
  • 다른 모든 요소들은 <html> 태그의 자식 요소이여야 합니다.

접근성(Accessibility)

  • <html> 태그에 lang 속성을 설정하면 화면 판독에 필요한 언어를 설정하는데 도움이 됩니다.
  • lang 속성은 웹 페이지에서 주로 사용하는 언어를 설정하며, 이 기능이 없으면 화면 판독기가 운영체에 언어를 기본 언어로 설정하여, 잘못된 언어를 표현할 수 있습니다.

속성(Property)

속성 값(예) 설명 버전
manifest manifest = "URL" 리소스를 나타내는 URL을 설정합니다.
version version="html5" 문서의 버전을 설정합니다. 문서 타입과 중복되기 때문에 현재는 사용하지 않습니다.
xmlns xmlns="http://www.w3.org/1999/xhtml" XML 네임스페이스 주소를 설정합니다. -
