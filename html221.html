HTML

HTML은 웹 문서를 표현하기 위한 하이퍼 텍스트 마크업(Hyper Text Markup Language) 언어입니다.

HTML 버전 및 종류

  • XML : XML은 'eXtensible Markup Language'의 약자로 HTML을 개선하여 만든 확장 가능 마크업 언어입니다.
  • XHTML : XHTML은 'eXtensible Hypertext Markup Language'의 약자로 XML과 HTML을 이용한 마크업 언어입니다.
  • HTML5 : HTML5는 HTML4와 XHTML1.0의 다음 버전입니다. 시멘틱 요소와 미디어 요소를 제공하며 다양한 API를 제공합니다.

HTML의 버전

1989년 팀 버너스리에 의해 HTML이 처음 만들어졌습니다. 처음에는 문서와 문서를 서로 이동하기 위해 만들어 졌으며, 팀 버너리스는 이 아이디어를 통해 월드 와이드 웹(World Wide Web) 개념을 개발에 1991년에 배포했습니다.
