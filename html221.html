HTML

HTML은 'HyperText Markup Language'의 약자로 하이퍼텍스트를 표현하기 위한 마크업 언어입니다.

하이퍼텍스트란 웹 사이트에서 링크를 통해 문서나 다른 문서로 이동하는 기능을 말하며, 마크업이란 태그를 사용하여 문서를 작성하는 것을 말합니다.

  • XML : XML은 'eXtensible Markup Language'의 약자로 HTML을 개선하여 만든 확장 가능 마크업 언어입니다.
  • XHTML : XHTML은 'eXtensible Hypertext Markup Language'의 약자로 XML과 HTML을 이용한 마크업 언어입니다.
  • HTML5 : HTML5는 HTML4와 XHTML1.0의 다음 버전입니다. 시멘틱 요소와 미디어 요소를 제공하며 다양한 API를 제공합니다.

HTML의 버전

1989년 팀 버너스리에 의해 HTML이 처음 만들어졌습니다. 처음에는 문서와 문서를 서로 이동하기 위해 만들어 졌으며, 팀 버너리스는 이 아이디어를 통해 월드 와이드 웹(World Wide Web) 개념을 개발에 1991년에 배포했습니다.

HTML의 분류

콘텐츠 카테고리(Content categories)

HTML 태그를 콘텐츠의 종류에 따라 Metadata Content, Flow Content, Section Conetnt, Heading Content, Phrasing Content, Embedded Content, Interactive Content, Palpable Content를 적용 할 수 있습니다.

콘텐츠 속성(Content properties)

HTML 태그에 글로벌 속성과 이벤트 속성을 적용 할 수 있습니다.

콘텐츠 요소(Content element)

HTML 태그를 요소에 따라 블록요소와 인라인 요소로 적용할 수 있습니다.
